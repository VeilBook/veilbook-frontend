--- PAGES TO BUILD ---

1. / (Homepage)
- Hero section with tagline and connect wallet button
- 3 feature cards: Confidential Orders, Zero MEV, Peer-to-Peer Settlement
- No contract calls needed

2. /faucet
- Show 5 token cards: USDC, NBL, SLR, ATH, VTX
- Each card: token name, symbol, user balance (call balanceOf)
- Mint button calls token.mint(userAddress, amount) — mint 1000 tokens
- USDC has 6 decimals (mint 1000 * 10^6), all others have 18 decimals (mint 1000 * 10^18)
- Show tx pending/success states

3. /trade
- Pool selector dropdown using POOL_KEYS (display as "NBL/USDC", "SLR/USDC" etc)
- Fetch and display current tick and price using StateView.getSlot0(poolId)
  Compute poolId with ethers.js:
  const poolId = ethers.keccak256(ethers.AbiCoder.defaultAbiCoder().encode(
    ['address','address','uint24','int24','address'],
    [key.currency0, key.currency1, key.fee, key.tickSpacing, key.hooks]
  ))
- Direction toggle: Buy (zeroForOne=false) / Sell (zeroForOne=true)
  Since USDC is currency1:
    Buy  = zeroForOne=false → spending USDC (currency1) to get token (currency0)
    Sell = zeroForOne=true  → spending token (currency0) to get USDC (currency1)
- Tick input (multiples of 60 only, validate on frontend)
- Amount input (plaintext — encrypted by Zama SDK before sending)
- Before placeOrder: user must approve encToken via encToken.setOperator(hookAddress)
  Get encToken address from VeilBook.getEncryptedToken(poolId, depositCurrency)
  depositCurrency = zeroForOne ? currency1 : currency0
- Place Order calls VeilBook.placeOrder(poolKey, tick, zeroForOne, encAmountIn, proof)
  Use Zama fhEVM SDK to encrypt amount before sending

4. /orders
- Call VeilBook.getUserOrders(userAddress) to get bytes32[] of order IDs
- For each orderId call VeilBook.getOrder(orderId) to get order details
- Show table: Pool, Tick, Direction (Buy/Sell), Status (Active/Filled/Cancelled), Actions
- Encrypted amounts shown as "🔒 Encrypted" with a decrypt button using Zama Relayer
- Cancel button calls VeilBook.cancelOrder(orderId, plaintextUnfilled)
  User must decrypt filledIn first, then unfilled = amountIn - filledIn
- Claim button calls VeilBook.claimFill(orderId, plaintextFilled)
  User must decrypt filledOut first

5. /pools
- 4 pool cards: NBL/USDC, SLR/USDC, ATH/USDC, VTX/USDC
- Each card fetches StateView.getSlot0(poolId) → show sqrtPriceX96, current tick, lpFee
- Convert sqrtPriceX96 to human readable price:
  price = (sqrtPriceX96 / 2^96)^2, adjust for decimals (token=18, USDC=6)
  adjusted price = raw price * 10^(18-6) = raw price * 10^12

--- IMPORTANT NOTES ---
- USDC is always currency1 across all pools (highest address)
- Token0 is always the non-USDC token (NBL, SLR, ATH, VTX) with 18 decimals
- USDC has 6 decimals, all other tokens have 18 decimals
- Tick must always be a multiple of tickSpacing (60) — round on input
- zeroForOne=true  → selling currency0 (the token) for USDC
- zeroForOne=false → selling currency1 (USDC) for the token
- depositCurrency for an order = zeroForOne ? currency1 (USDC) : currency0 (token)
- PoolId = keccak256(abi.encode(currency0, currency1, fee, tickSpacing, hooks))
- For Zama FHE encryption use @zama-ai/fhevm-js SDK
- Handle wallet not connected state gracefully on all pages
- All transactions show pending → success → error toast notifications

// ── Helper: encode PoolKey ────────────────────────────────────────────────────
function encodePoolKey(
  currency0: string,
  currency1: string,
  fee: number,
  tickSpacing: number,
  hooks: string
) {
  return { currency0, currency1, fee, tickSpacing, hooks };
}

// ── Helper: sort token addresses ─────────────────────────────────────────────
function sortTokens(a: string, b: string): [string, string] {
  return a.toLowerCase() < b.toLowerCase() ? [a, b] : [b, a];
}

// ── Helper: pool ID from key ──────────────────────────────────────────────────
function getPoolId(key: ReturnType<typeof encodePoolKey>): string {
  return ethers.keccak256(
    ethers.AbiCoder.defaultAbiCoder().encode(
      ["address", "address", "uint24", "int24", "address"],
      [key.currency0, key.currency1, key.fee, key.tickSpacing, key.hooks]
    )
  );
}

function tickToPrice(tick: number): number {
  return Math.pow(1.0001, tick);
}

function tickToSqrtPriceX96(tick: number): bigint {
  const price = Math.pow(1.0001, tick);
  const sqrtPrice = Math.sqrt(price);
  return BigInt(Math.floor(sqrtPrice * 2**96));
}
